## Built With

- Reactjs
- Css
- Npm


## Live Demo 

you can check the live preview in [Here](https://space-travelers-hub.herokuapp.com/)


## Install

- first, clone the project
  Run

```
  git clone https://github.com/ichala/space-travelers-hub.git
```

- Install the necessary dependancies
  Run

```
  npm install
```

```
  npm start
```
 ## Test
 
```
  npm test 
```
